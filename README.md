# Mobile-Sales-Dashboard-BI

1. Project Title / Headline

Mobile Sales Dashboard using Power BI
An interactive business intelligence dashboard built to analyze mobile sales performance, customer trends, revenue insights, and product-wise sales data using Power BI.

2. Short Description / Purpose

This project focuses on creating a dynamic and interactive Mobile Sales Dashboard using Power BI. The dashboard helps visualize key business metrics such as total sales, top-performing mobile brands, customer purchasing behavior, and regional sales analysis.
The main objective of this project is to transform raw sales data into meaningful insights through data visualization and analytics. It enables better understanding of sales trends and supports data-driven decision making.

3. Tech Stack
Power BI – Data visualization & dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Calculated measures and KPIs
Microsoft Excel / CSV – Dataset handling


4. Data Source
The dataset used in this project was collected from an online YouTube tutorial source for educational and learning purposes.

5. Features / Highlights

Business Problem-
Mobile sales businesses generate a huge amount of sales data every day, making manual analysis difficult and time-consuming. Without proper visualization, it becomes challenging to identify top-selling products, understand customer purchasing patterns, track regional performance, and monitor overall business growth effectively.

Goal of the Dashboard-
The primary goal of this dashboard is to transform raw mobile sales data into meaningful business insights through interactive visualizations. The dashboard helps users monitor sales performance, analyze revenue trends, compare mobile brand sales, and make better data-driven business decisions quickly and efficiently.

Key Visuals
* KPI cards displaying total sales, revenue, quantity sold, and overall performance metrics
* Bar charts for comparing sales across different mobile brands and models
* Pie and donut charts showing category-wise and regional sales contribution
* Line charts to visualize monthly and yearly sales trends
* Region-wise sales analysis to identify high-performing areas
* Interactive filters and slicers for detailed and customized data analysis

Business Impact & Insights-
This dashboard helps businesses improve decision-making by providing clear insights into sales performance and customer trends. It enables faster reporting, identifies best-selling mobile brands and products, supports better inventory and marketing strategies, and improves overall business efficiency through data-driven analysis.
